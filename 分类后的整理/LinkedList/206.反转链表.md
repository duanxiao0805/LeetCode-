### 反转一个单链表
##### 实例
输入:
```in
1->2->3->4->5->null
```
```out
5->4->3->2->1->null
```
```Java
public class ListNode {
      int val;
      ListNode(){}
      ListNode(int val){this.val=val;}
      ListNode(int val,ListNode next){this.val=val;}
}

  public ListNode reverseList(ListNode head){
```
