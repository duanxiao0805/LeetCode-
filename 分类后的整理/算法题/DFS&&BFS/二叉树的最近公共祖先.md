递归子函数上面的代码,写单层逻辑

递归子函数下面的代码,可以理解为已经全部回到了起初刚开始的位置,也就是root位置,所以这个时候的函数结点状态root,只是left和right进行了判断而已

```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */
class Solution {
    public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
        if(root==null)
            return null;
        //查找这个结点
        if(root==p||root==q)
            return root;
        //递归子函数
        TreeNode left=lowestCommonAncestor(root.left,p,q);
        TreeNode right=lowestCommonAncestor(root.right,p,q);
        
        //已经全部回到第一层,这个时候想的话,就直接想成三个结点的数就可以了
        //此时的状态---root,left,right
        if(left==null)
            return right;
        if(right==null)
            return left;
        if(left!=null&&right!=null)
            return root;
        
        return null;
        
    }
}
```
