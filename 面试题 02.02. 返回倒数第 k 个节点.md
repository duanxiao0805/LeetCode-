### 实现一种算法，找出单向链表中倒数第 k 个节点。返回该节点的值。

注意：本题相对原题稍作改动

示例：

输入： 

```in
1->2->3->4->5 和 k = 2
```

输出：

```out
4
```

```Java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) { val = x; }
 * }
 */

//思维别乱套,head是一系列的数据,后面连着呢!

class Solution {
    public int kthToLast(ListNode head, int k) {
        //用笨办法
        //求出链表的长度
        int len=0;
        ListNode temp=head;
        while(temp!=null){
            temp=temp.next;
            len++;
        }
        for(int i=0;i<len-k;i++){
            head=head.next;
        }
        return head.val;
    }
}
```
