### 给你一个整数数组 nums，请你将该数组升序排列。

**示例 1：**

输入：
```
nums = [5,2,3,1]
```
输出：
```
[1,2,3,5]
```

**示例 2：**

输入：
```
nums = [5,1,1,2,0,0]
```
输出：
```
[0,0,1,1,2,5]
```

**思路:**

这里是用的交换的方法

```java
class Solution {
    public int[] sortArray(int[] nums) {
            quickSort(nums,0,nums.length-1);
        return nums;
    }
    
    public void quickSort(int []arr,int left,int right){
        if(left>right)
            return ;
        //定义变量保存基准数
        int base=arr[left];
        //定义变量i,指向最左边
        int i=left;
        //定义变量j,指向最右边
        int j=right;
        
        //当i和j不相遇的时候,在循环中进行检索
        while(i!=j){
             while(arr[j]>=base&&i<j) {
                 j--;   //j向左移动
             }
            while(arr[i]<=base&&i<j) {
                i++;    //i向右移动
            }
            //代码走到这里 i停下了,j页停下了
            //然后交换i和j位置的元素
            int temp=arr[i];
            arr[i]=arr[j];
            arr[j]=temp;
        }
        //把相遇位置的元素赋值给基准数这个位置的元素
        arr[left]=arr[i];
        arr[i]=base;
        
        //递归调用
        quickSort(arr,left,i-1);
        quickSort(arr,j+1,right);
    }
}
```
