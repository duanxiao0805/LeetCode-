### 给你一个链表，删除链表的倒数第 n 个结点，并且返回链表的头结点。

**示例 1：**
![image](https://user-images.githubusercontent.com/62934005/117562617-dfd79380-b0d2-11eb-9413-565f1f8b5b06.png)

输入：
```
head = [1,2,3,4,5], n = 2
```
输出：
```
[1,2,3,5]
```
