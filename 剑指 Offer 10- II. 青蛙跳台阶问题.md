### 一只青蛙一次可以跳上1级台阶，也可以跳上2级台阶。求该青蛙跳上一个 n 级的台阶总共有多少种跳法。



示例 1：

输入：

```
n = 2
```

输出：

```
2
```


示例 2：

输入：

```in
n = 7
```


输出：

```out
21
```


示例 3：

输入：

```in
n = 0
```


输出：

```out
1
```
```Java
//斐波那契数列应用
class Solution {
    public int numWays(int n) {
        int a=0;    //第一个数a
        int b=1;    //第二个数b
        if(n==0||n==1){
            return 1;
        }
        //取余
        while(n>0){
            int temp=(a+b)%1000000007;
            a=b%1000000007;
            b=temp%1000000007;
            n--;
        }
        return b%1000000007;
    }
}
```
