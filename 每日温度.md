**单调栈的应用:**

```java
class Solution {
    public int[] dailyTemperatures(int[] T) {
        //放入第几天
        int len=T.length;
        Stack<Integer> st=new Stack<>();
        int[] res=new int[len];
        for(int i=0;i<len;i++){
            while(!st.isEmpty()&&T[i]>T[st.peek()]){
                res[st.peek()]=i-st.peek();
                st.pop();
            }
            st.push(i);
        }
        return res;
    }
}
```
